# IROFIT — placeholder site

Premium fitness lifestyle brand. Three energies, one rhythm.

## Stack
- Vite + React 18
- Tailwind CSS
- framer-motion
- react-router-dom

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Pages
- `/` — Home (animated hero, multilingual coming soon, three energies, CTA)
- `/about` — Brand story
- `/contact` — Contact + email capture (local placeholder)
